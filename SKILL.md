# Testing Skill

Use this skill when a developer asks for help with testing, unit tests, integration tests, or quality checks.

## When to use

- Writing or improving tests
- Understanding test structure
- Finding existing testing patterns
- Debugging failed tests

## Guidance

Prefer existing project test patterns before introducing new tools or conventions.
